# mate

Script and readme in german only, sorry. ;-)

Ein Skript, das sich meinen Kontostand bei der Getränkekasse in unserem Club-Raum merkt. Nur eine kleine Spielerei. :)

Beispiele:
  $ mate
  > Kontostand: 2,30€

  $ mate +3
  > 3,00€ eingezahlt.

  $ mate -1,5
  > 1,50€ abgezogen.

  $ mate -
  > 1,20€ abgezogen.
